# 보안 정책 (Security Policy)

## 취약점 제보

보안 취약점을 발견하면 **공개 이슈로 올리지 말아주세요.** 악용을 막기 위해 비공개로 먼저 알려야 합니다.

- GitHub **Security Advisory**(저장소 → Security → "Report a vulnerability") 또는
- 동아리 단톡방의 운영진, 또는 회장·교육부장에게 개인적으로 연락(공개 단톡방에 올리지 말 것)

가능하면 다음을 포함해주세요: 재현 방법, 영향 범위, 가능하면 수정 제안.

확인 후 영업일 기준 며칠 내 회신하며, 수정 전까지 비공개로 유지합니다.

## 범위

이 프로젝트는 인증(GitHub OAuth)·파일 업로드·`.env` 시크릿을 다룹니다. 시크릿은 절대 커밋하지 않으며(`.env.example`만), 노출된 키를 발견하면 위 경로로 알려주세요.
